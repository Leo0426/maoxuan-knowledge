# MaoXuan Knowledge
> ps: 个人学习，强化记忆项目，出发点是可以记住强化认知；

《毛泽东选集》知识可视化系统，用时间线、地图、知识图谱、文章目录和检索，把文本阅读变成一个可探索的档案工作台。

## 功能概览

- 导读：四卷内容概览与阅读线索
- 文章：按卷、历史时期、篇目浏览原文
- 时间线：按文章查看关键事件，支持事件定位、缩放和详情联动
- 地图：中国地图上按地点标注事件，支持时间筛选
- 知识图谱：展示文章、事件、思想、实体之间的关系
- 搜索：检索文章、事件、思想和实体
- 问题反馈：通过 `mailto:` 生成邮件草稿
- 免责声明：说明资料、抽取结果和使用边界
